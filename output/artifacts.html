This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide.

Mental Health Improvement Status

A profile to capture the improvement status of a patient’s mental health condition.
Mental Health Patient

A FHIR Patient profile tailored for mental health contexts in Nepal.
Mental Health Referral Request

A profile for capturing referrals to mental health services, such as psychiatry, including clinical justification.
Mental Health Treatment Follow-up Status

A profile to capture a patient's compliance or follow-up status in ongoing mental health treatment.
Primary Diagnosis Condition with any ICD-11 Mental Health Code

A Condition resource representing any primary mental health diagnosis from ICD-11 codes.

Structures: Extension Definitions

These define constraints on FHIR data types for systems conforming to this implementation guide.

Ethnic Code

Extension to capture जाति कोड (ethnicity) based on Nepali classification

Terminology: Value Sets

These define sets of codes used by systems conforming to this implementation guide.

Ethnic Code ValueSet

ValueSet for जाति कोड (Ethnic Codes in Nepal)
Improvement Status ValueSet

Allowed values for improvement status.

Codes representing the subjective improvement status of the patient.
